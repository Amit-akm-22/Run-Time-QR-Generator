Create flawless, scannable UPI payment QR codes instantly directly in your browser. This application provides a beautifully crafted premium dark mode interface with offline capabilities — your data is never sent to any server.

---

### ✨ Features
- **Instant Generation**: Generate codes by simply typing a UPI ID and (optional) payee name or amount.
- **Deep-Link Ready**: Creates a compliant `upi://pay` deep link that launches directly into any UPI app (GPay, PhonePe, Paytm).
- **One-Click Download**: Instantly export a high-quality, perfectly squared PNG of your QR Code.
- **Ultra-Premium UI**: Glassmorphic styling, ambient background glows, animated gradients, and 100% responsiveness.
- **Micro-Interactions**: Features a dedicated toast notification system, quick-copy to clipboard logic, and interactive hover states.

<br />

### 🛠 Tech Stack & Tools Used
Built as a powerful **Single-File React Application**, utilizing no build steps for pure simplicity and portability.

* **HTML5 & CSS3**: Core structural and visual framework with advanced CSS clamp scaling and CSS variables.
* **React 18**: Dynamic UI state management (served via CDN).
* **Babel Standalone**: In-browser JSX transpilation (served via CDN).
* **QRCode.js**: Lightweight dependency to natively parse strings into an onscreen QR grid/canvas.

<br />

### 🚀 Getting Started
Because this app is entirely self-contained, there are no packages to download or dependencies to build!

1. Clone or download the repository.
2. Open `index.html` in your favorite web browser.
3. *That's it!* Begin generating payment stickers immediately.
